Правила REVOLUTION

Обязательны к соблюдению всеми игроками и администрацией сервера.

❗ Общие правила для игроков

1. Запрещены оскорбления, унижение игроков, токсичное поведение и чрезмерный мат. 2. Запрещён флуд в чат, микрофон или радио-команды. 3. Запрещена реклама в любом виде. 4. Запрещены читы, скрипты, баги и любые сторонние программы. 5. Запрещено мешать нормальной игре другим игрокам. 6. Обсуждение наказаний — только с администрацией.

❗ Наказания

1. Флуд / мат — 3 предупреждения → мут 30 минут 2. Реклама — мут / смена ника 3. Оскорбления — 2 предупреждения → мут 1 час 4. Читы — бан НАВСЕГДА 5. Повторные нарушения — увеличение наказания

❗ Правила для администраторов

1. Администратор обязан соблюдать правила сервера. 2. Запрещено использовать админку в личных целях. 3. Запрещены баны без причины. 4. Запрещено превышение сроков наказаний. 5. Карты меняются только по правилам сервера. 6. Запрещена передача админки. 7. Запрещён слив паролей.

❗ Незнание правил не освобождает от ответственности
